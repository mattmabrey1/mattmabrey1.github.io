Latest Projects

- Demolition Derby Game [C#, Unity Engine, Blender, Photon Networking Framework]

Independent project where each player controls a car and one randomly assigned player gets a bomb with a timer. The player with the bomb must hit another car before the time runs to transfer the bomb or they will explode. The game goes until only one player remains

- "Would You Rather...?" Website [HTML, CSS, Javascript, Brackets]

Final project for "Fundamentals of Web Design" course at TCNJ during the Fall 2019 semester. The website allows the user to play a game of would you rather, submit questions of their own, sort questions, and like/dislike/comment on questions

- Hunter, Fox and Rabbit Population Simulation [Java, BlueJ Object-Oriented Programming IDE]

Final project for "Objected-oriented Programming & Data Abstraction" course at Rowan College at Burlington County during the Spring 2019 semester. The program simulates the population of hunters, foxes, and rabbits in a 120x80 grid. Each "actor" in the simulation has specific actions such as moving, killing, eating, reproducing, and dying.

- Hangman Game [Assembly Language, Microsoft Macro Assembler(MASM)]

Final project for Machine and Assembly Language programming course at Rowan College at Burlington County during the Spring 2019 semester. The program allows the user to play a game of hangman, keeping track of the word to solve, previously guessed letters, and the body parts of the hangman
